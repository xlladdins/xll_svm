# xll_svm

An Excel add-in for [libsvm](https://www.csie.ntu.edu.tw/~cjlin/libsvm/).

Support Vector Machines are a generalization of perceptrons that can maximize the margin of the separating hyperplane, among other things.

Every `model` requires a `problem` consisting of data and `parameter`s indication the type of the model and how it trains the data.
